# Практическая работа №7.3

## Описание
В этой работе реализован пример Android-приложения, отправляющего системные уведомления двух типов.

## Что было сделано
1. На основной экран добавлены кнопка с изображением и отдельная кнопка для уведомления с прогрессом.
2. Создан канал уведомлений для устройств с Android 8 и выше.
3. Настроено стандартное уведомление с крупной иконкой, заголовком и текстом.
4. Реализован второй вариант уведомления с бесконечным индикатором выполнения.
5. Добавлена обработка разрешения `POST_NOTIFICATIONS` для Android 13+.
6. Настроены приоритет, автоудаление и видимость уведомлений.

## Результат
Проект демонстрирует базовую работу с `NotificationChannel`, `NotificationCompat` и разрешениями для показа уведомлений.
