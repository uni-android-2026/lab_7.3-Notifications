# Практическая работа №7
## Задание: Разработка приложения
Концепция приложения: работа с уведомлениями Android.

## Проделанная работа:
1. Создан UI с кнопкой-колокольчиком и кнопкой показа уведомления с процессом.
2. Настроен канал уведомлений для Android 8+ и параметры приоритета/видимости.
3. Реализовано обычное уведомление с большой иконкой (`bell_large`) и автоудалением при клике.
4. Реализовано уведомление с бесконечным прогрессом и флагом `ongoing`.
5. Добавлена проверка и запрос разрешения `POST_NOTIFICATIONS` для Android 13+.

## Итоги
Приложение демонстрирует создание каналов и двух типов уведомлений, а также работу с разрешениями.
